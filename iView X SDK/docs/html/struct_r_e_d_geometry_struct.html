Detailed Description

Use this struct to customize the RED and RED-m geometry.

See chapter Setting up RED and RED-m Geometry in the iView X SDK Manual for details. For setting up the RED or RED-m geometry parameters with REDGeometryStruct use function iV_SetREDGeometry.
